真情流露：telnet://bbs.crspd.bupt.edu.cn

　　这个ＢＢＳ是北京邮电大学的，它有着一个特殊的妙用，呵呵。

　　如果你在北京邮电大学中上网的话，你就可以使用一个免费的，公开的代理器(Proxy Server)。这个代理器并不是平时我们常用来免费出国之类的代理器，而是一个“提速”的代理器。它支持 Telnet，Http。

　　怎样提速呢？提什么速呢？首先要说中国国内的网络速度。因为中国内部的几个破网之间用来通信的带宽非常的少，所以像我酱紫的在教育网内的学生要是想联到 ChinaNet 中，速度会非常的慢：慢到什么程度呢？无法忍受。而北邮这个代理服务器连教育网外面的网络是非常的快（比现在四网互连后也快），所以北邮里面的大虾可能都非常爽吧！

　　遗憾的是，我们没有办法直接使用这个代理服务器，因为它的 IP 是形如：10.6.10.10 的，只有在北邮内部才能连的上。:~( 我们只能通过真情流露的银河穿梭（BBS Net）来实现。不过酱紫我们只能通过这个代理去像广州蓝天ＢＢＳ酱紫的非教育网ＢＢＳ了。

　　具体使用？你只要注册成功一个属于你自己的 ID ，然后用真情流露的服务项目里的 BBS Net(银河穿梭) 就可以了。呵呵，在银河穿梭中好象是第三项：Ritt Proxy Server ，一连就跑到了 ... ... 香山红叶站。这个香山红叶站就是这个 10.6.10.10 ，可以通过它的 BBS Net 连到 ChinaNet 中的 ＢＢＳ ，而 ChinaNet 中的ＢＢＳ又提供大量的直接银河穿梭，几乎包括了中国所有的 ＢＢＳ（包括香港和台湾） 。

　　具体效果么，我记得在没有四网互连的时候，直接 Ping 广州蓝天ＢＢＳ（bbs.gznet.com）是 Time Out ，也就是不通。而通过真情流露穿梭到 ritt Proxy 在穿梭到佛山ＢＢＳ又穿梭到广州的蓝天ＢＢＳ，速度比直接连水母青蛙还要快。
