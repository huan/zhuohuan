Netterm 是我认为的最好的上ＢＢＳ的软件。它功能强大，界面美观，支持脚本语言(Script，也就是像用 ZMud 做的机器人，可以做灌水机，可以自动登陆:-)，支持当别人给你发信息的时候响铃提示。选项非常多，只要设置好了，几乎可以使任何人满意。^_^

　　我为大家介绍一下我的经验：

有人说 Netterm 的界面非常恶心，这是因为选项设置不正确造成的。在连接(Connect)之前，应该确认菜单中：File->Phone Directory->对话框中左下脚的Emulation选择框中为ANSI-BBS而不是ANSI。
ANSI为恶心的界面。
ANSI-BBS的界面看着非常爽。
在ＢＢＳ上收到 Message 信息的时候，Netterm 可以发出响声以提示你。具体做法：确认菜单 Options->Setup->Global Settings->[General标签中的 Bell on 为选择状态，Special 标签中的 If the bell option is select ... ... 中选 Exclamation Sound 为选择状态（没有声卡则选择 Speaker）]
如果想把自己浏览过的文章（在 Netterm 的缓冲区中）存盘的话，选择菜单项 Edit->Save Scroll Buffer As...
一些工具在菜单项 Options->Tools 中
Finger ... ：查询某个服务器上的用户的状态。
Resolve ...：解析域名或查询 IP 的域名。
This Host ...：显示本地主机的 IP 地址和 域名。
FTP Server ...：启动一个匿名 FTP 服务器。
如果用 Netterm 玩 MUD
不进行设置会出现 MUD 不认回车，必须 Ctrl+Enter 才可以。改变的方法：在工具条最右边的 Destop Settings 中 Return Sends 选 CRLF 而不是 CR。
输入的命令 Netterm 缺省是不显示。如果想要显示输入的命令，在工具条最右边的 Destop Settings 中 Line Control 中选 Local Echo。
可以在 Netterm 的下面加上一条像 ZMud 那样的输入条。做法：在工具条最右边的 Destop Settings 中选 Add input data window to the StatusBar。
使用输入条不过瘾的话可以专门分出一个窗口来管理输入。做法：选菜单项 Options->Tools->Floating Input。
