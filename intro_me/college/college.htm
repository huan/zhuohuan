在高中的时候，常常听老师说：“你们到了大学里就轻松了，大学里课程非常少，自己支配的业余时间很多 ... ... ”

　　带着一丝兴奋与神秘踏进了青蛙的大门的我，带着太多的幻想。报到时，经过了验证身份，医生盘问，领取日常用品，宿舍钥匙后，在我的下铺：彭剑锋（Mmwan冷剑）的带领下，我来到了12#316，也就是我大一的宿舍。我的父母也在，他们看起来还比较满意这里的环境。在这里，我陆续认识了我的其余四个舍友：任宇鹏（老人）、刘金国（zxcvb大酸酸）、姚为（姚夏）和仝文权（TongWen水王）。大学生活由此拉开了帷幕 ... ...

　　首先是两个星期的军训，也就是由一群军校的大学生来“处理”一下我们罢了。无所谓的说。接下来，破青蛙的紧张学业便紧紧的压在了我的身上 ... ... 微积分、线性代数、机械制图、大学化学 ... ... 棵棵都很不好学。我用我在高中学习所要求的熟练程度来要求自己（偶在高中其实也稀松平常），但是我达不到。所以有一段时间我很不习惯。唉，没有办法，谁叫自己脑子笨呢 ... ... 习惯了就好了，嘻嘻。

　　在青蛙里见识了一大批牛人。比如俺们班的瞿华（Roy游马），高考分数650分（对偶来说确实很高，但是假如您老是更牛的人，那俺就 Admire 先了。:-）呵呵，平时的学习对他来说简直是小意思，啧啧啧啧，为什么不是我 ... ... ;-P 还有计算机系的 Nolove 大虾，计算机那是“母牛做绝育手术──牛 X 死了” ^_^
